# FieldService Knowledge Base

This repository is a **canonical, public knowledge base** focused on AI, Field Service, and Copilot architecture.

It exists to provide **durable reference material**—frameworks, terminology, diagrams, and presentation assets—that can be cited, shared, and reused without link rot or ambiguity.

This is not a development repository.
It is a **curated library**.

---

## How to Use This Repository

Use this repository when you need:

* A stable reference to AI and Copilot concepts
* Clear terminology across vendors and platforms
* Architecture frameworks for Field Service and connected operations
* Downloadable decks and diagrams intended for external sharing

All content here is intentionally structured, versioned by time or context, and designed to remain link-stable.

---

## Governance Note 
**Governance note:** Issues, discussions, and forums are intentionally disabled—this repository is curated as a stable reference library, not a collaborative workspace or support channel.

----

## Repository Structure

* **/lexicon**
  Canonical terminology and vocabulary mappings

* **/frameworks**
  Architectural models, operating patterns, and reference designs

* **/decks**
  Published presentation assets (PowerPoint, PDF), organized by year

* **/diagrams**
  System diagrams, conceptual visuals, and architecture illustrations

* **/references**
  Supporting links, citations, and external source material

---

## Publishing Philosophy

* Content in this repository is considered **published**
* Files are not renamed once referenced publicly
* Updates are additive and time-scoped, not destructive
* Git history provides provenance and evolution

Drafts and experimental material live elsewhere.
